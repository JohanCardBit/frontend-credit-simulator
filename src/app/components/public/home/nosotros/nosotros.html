Portal de simulación de créditos y préstamos

  • Objetivo: permitir a los clientes simular préstamos por monto, plazo, tasa e perfil de riesgo.
  • Back-end: API para cálculos financieros (cuotas, tasas, amortizaciones) — Node.js + Express.
  • Front-end: Interfaz en Angular con formularios dinámicos y visualización (gráficas y tablas).
  • Extras: Validación de reglas de negocio y simulación de aprobación automática por perfil.

Tecnologías & despliegue

  • Angular
  • TypeScript
  • HTML 5 / SCSS
  • Bootstrap
  • Node.js
  • Express
  • MongoDB
  • AWS

Tipo: App sencilla pero moderna
